About This Project

Sign Language Converter is a real-time AI-powered application that detects American Sign Language (ASL) gestures using your webcam, converts them to text, translates them into Hindi (and other Indian languages), and speaks the translated text.


Technology Stack:

  • MobileNetV2 (Deep Learning Model)
  • TensorFlow / Keras
  • OpenCV for real-time video processing
  • Google Translate API for translation
  • Google Text-to-Speech (gTTS) for Hindi speech
  • Flask for Web Backend
